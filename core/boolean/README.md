# [Boolean](http://php.net/manual/en/language.types.boolean.php)

```php
$variable = true;
var_dump($variable); //=> bool(true)

$variable = True;
var_dump($variable); //=> bool(true)

$variable = TRUE;
var_dump($variable); //=> bool(true)
```

```php
$variable = false;
var_dump($variable); //=> bool(false)
```
